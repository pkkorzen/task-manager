The app allows to view all the tasks, mark them as finished/unfinished and view only finished/unfinished taks. You can also easily add, edit and remove your tasks.

Tech stack:
- Java 8,
- Spring boot 2.0.6,
- gradle,
- H2 database,
- hibernate 5.2.17,
- bootstrap 3.3.7,
- thymeleaf,
- Tomcat 8.5.34
